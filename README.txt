THE WORLD - DotHack React Native

Projeto inspirado em .Hack desenvolvido em React Native usando Expo Router.

Funcionalidades

- Tela de Boot
- Login
- Menu Principal
- Sistema de Mail
- Tela de Personagens
- Tela de Configurações
- Simulação de Gameplay com vídeo
- Reprodução de música

Tecnologias

- React Native
- Expo
- Expo Router

 Instalação

```bash
npm install
npx expo start
```
